Click to expand: What is EdibleCSS?

EdibleCSS is a primitive, classless CSS framework for developers with no design skills. It styles semantic HTML5 elements with a single link tag, requiring zero CSS classes or JavaScript.

Code Example

Here's how to use EdibleCSS:

<!DOCTYPE html>
<html lang="en">
<head>
  <link rel="stylesheet" href="edible.css">
</head>
<body>
  <h1>Hello World</h1>
  <p>Your styled page!</p>
</body>
</html>

FAQ 1: How do I install EdibleCSS?

Add a single link tag: <link rel="stylesheet" href="edible.css">

FAQ 2: Do I need to write CSS classes?

No! EdibleCSS styles semantic HTML elements only. Zero classes required.

FAQ 3: Is JavaScript required?

No JavaScript needed. Pure CSS solution.

FAQ 4: What browsers are supported?

Modern evergreen browsers: Chrome, Firefox, Safari, and Edge (last 2 versions).
